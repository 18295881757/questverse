# QuestVerse 30-45 分钟试玩节奏

> 目标：验证 Phase 2 MVP 是否能让首次玩家完成两颗星球的核心闭环，并在结束时自然理解第三颗星球的方向。
> 版本：Phase 2 / 2026-07-04

---

## 1. 试玩目标

- 玩家能从主页进入星球选择，并理解第一颗星球是起点。
- 玩家能完成铜钥匙闭环：街机厅探索 → 逆向谜题 → 隐藏房间 → Reverie 第一段线索。
- 玩家能进入黑胶唱片室，完成玉钥匙闭环：倒放标签 → 频率寻找 → 唱片排序 → 声纹抽屉 → 磁带档案室。
- 玩家能在档案页看到自己的钥匙、便签和彩蛋，理解“钥匙是节点，便签是叙事”。
- 玩家在结束时能说出第三颗星球的伏笔：被剪掉的 3 分 17 秒影像。

---

## 2. 推荐试玩流程

| 时间 | 流程 | 设计意图 | 验收观察 |
|---|---|---|---|
| 0-3 分钟 | 主页 → 星球选择 → 街机厅入口 | 建立项目气质和入口路径 | 玩家是否能找到“进入游戏” |
| 3-8 分钟 | 探索街机厅入口与主厅 hotspots | 让玩家意识到彩蛋密度 | 玩家是否会主动点海报、机台和角落 |
| 8-13 分钟 | 完成 `puzzle_atari_secret_room` | 建立“逆向思维”规则 | 玩家是否理解倒序输入，而不是乱点 |
| 13-18 分钟 | 进入隐藏房间，读取终端、便签、系统地图 | 把第一钥匙转成第二星球线索 | 玩家是否会打开档案页确认收获 |
| 18-22 分钟 | 进入黑胶唱片室入口，观察票根与海报 | 切换星球主题 | 玩家是否理解“噪声/实体媒介”主题 |
| 22-28 分钟 | 试听室：倒放标签 + Echo-7 接收器 | 角色化提示和方向提示 | 玩家是否会用 `REWIND` 解码 |
| 28-34 分钟 | 遗失的频率隐藏对象谜题 | 验证勇气类谜题和失败容忍 | 玩家失败后是否愿意重试 |
| 34-39 分钟 | 唱片排序谜题 | 拉长第二星球并验证逻辑反馈 | 玩家是否使用 Echo-7 提示逐步修正 |
| 39-43 分钟 | 声纹抽屉 + 磁带档案室 + 剪帧索引 | 收束第二星球，指向第三星球 | 玩家是否能复述 `00:03:17:REVERIE` |
| 43-45 分钟 | 记忆档案页回看 | 验证持久化和叙事归档 | 档案页是否准确展示钥匙、便签和彩蛋 |

---

## 3. 观察重点

### 3.1 内容密度

- 第一星球入口、主厅、隐藏房间是否有足够可点内容。
- 玩家是否觉得非钥匙线索也有意义。
- 新增彩蛋是否能让玩家愿意继续探索，而不是只追主线。

### 3.2 谜题节奏

- 铜钥匙谜题应在 3-6 分钟内完成。
- 倒放标签应在 1-3 分钟内完成。
- 遗失的频率可以允许一次失败，失败后重试不应产生挫败。
- 唱片排序应允许玩家通过 Echo-7 提示完成，错误反馈必须指向“年份顺序”。

### 3.3 可访问性

- 所有主要按钮应能通过 Tab 聚焦。
- Enter 应能触发链接、热点按钮、谜题按钮和提交按钮。
- Phase 2 只把桌面网页端作为验收范围。
- 移动端完整适配与完整无障碍专项记录在 `docs/final-backlog.md`，不阻塞当前 demo。

---

## 4. 通过标准

Phase 2 试玩通过需要满足：

- 首次玩家能在 45 分钟内完成两颗星球主要闭环。
- 玩家至少获得 2 个钥匙碎片、4 条以上 Reverie/Echo 线索。
- 玩家至少发现 2 个彩蛋。
- 玩家不会因为失败状态被卡死。
- 桌面网页端不存在阻断性问题。
- 移动端和完整键盘/无障碍专项已转入最终保留项。
- 试玩结束后，玩家能自然说出下一阶段开发方向：影像星球、水晶钥匙、被剪掉的 3 分 17 秒。
