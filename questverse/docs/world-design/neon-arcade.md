**2041 年**。在云端雪崩之前，这家街机厅位于美国俄亥俄州哥伦布市的一个偏远小镇。

## 4. 核心谜题：雅达利：隐藏的房间

### 4.1 谜题设计

**类比**: 玩家在《Adventure》(Atari 2600, 1980) 中需要进入一个**隐藏的房间**——这个房间在游戏本身里**不存在**。

**叙事包装**:
> "这台街机屏幕还亮着——你看见它显示了《Adventure》游戏的画面。  
> 但地图边界外有什么？试着找到 Warren Robinett 留给玩家的话。"

**答案机制**:
- 玩家在谜题界面中看到《Adventure》游戏的简化像素地图
- 玩家可以"逆向"移动 — 不是走到目的地，而是从**已知区域返回**
- 关键动作：走到"地图边缘"——但屏幕**不滚动**（致敬 1979 年 Atari 2600 的限制）
- 然后玩家触发一个"闪屏"，显示："CREATED BY WARREN ROBINETT"（致敬第一个彩蛋）

**为什么这是"逆向思维"**:
- 玩家第一个直觉是"走到秘密区域"
- 真正答案是"承认游戏地图的**局限性**"，从局限中突破

### 4.2 谜题配置

```typescript
{
  id: "puzzle_atari_secret_room",
  type: "reverse",  // 铜钥匙
  name: "雅达利：隐藏的房间",
  hint: "游戏史上第一个 Easter Egg 藏在反向之中。Warren Robinett 把自己的名字藏在《Adventure》（雅达利 2600, 1980）的隐藏房间中。",
  hints: [
    "提示 1: 尝试逆向操作。",
    "提示 2: 真正的解法不需要赢，只需要进入特定的房间。",
    "提示 3: 游戏作者 Warren Robinett 希望玩家知道这是他自己做的 — 当游戏是 'Atari Adventure' 时，唯一能让其被记住的方法是留下名字。"
  ],
  maxHints: 3,
  sequence: ["UP", "UP", "DOWN", "DOWN", "LEFT", "RIGHT", "LEFT", "RIGHT"],
  items: [...],
  reverseExecution: true,
  reward: {
    itemId: "key_fragment_atari",
    storyProgress: 25
  },

}
```

### 4.3 解决后

玩家获得**铜钥匙碎片**：

> *Atari 钥匙碎片* — 来自 Warren Robinett 的《Adventure》  
>  
> "这枚碎片象征着'逆向思维'。Warren Robinett 在公司不让他署名的时代，用隐藏的名字赢得了永恒的纪念 — 这是游戏作者对游戏玩家的第一封情书。"

**叙事事件**:
- 故事进度 +25
- 触发 flag: `met_first_master` (致敬头号玩家的第一次会面)
- 玩家被传送到**记忆档案页面**，看到自己刚获得的物品
