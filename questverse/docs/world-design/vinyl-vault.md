# 第二颗星球：黑胶唱片室 (Vinyl Vault)

> 主题：80 年代音乐、实体媒介、在噪声中面对恐惧  
> 钥匙：玉钥匙（勇气）  
> 致敬对象：Joy Division、Kraftwerk、倒放磁带、黑胶收藏文化

---

## 1. 视觉定位

### 1.1 调色板

| 角色 | 颜色 | 用途 |
|---|---|---|
| 主色 | 紫罗兰 `#B14DFF` | 神秘感、唱片室门牌、互动边框 |
| 辅色 | 暖黄 `#F9E547` | 唱针、频率碎片、奖励 |
| 危险色 | 霓虹粉 `#FF2E88` | 噪声压力、倒计时 |
| 背景 | 深空蓝 / 午夜紫 | 地下唱片室、低照度空间 |

### 1.2 美术风格

- 地下唱片室、低天花板、厚重吸音墙；
- 黑胶圆形元素反复出现，形成“眼睛”般的压力感；
- 灯光不稳定，像音量电平表一样跳动；
- 使用 CRT/扫描线叠加，让空间像旧音乐录像带。

## 2. 故事设定

玩家带着第一颗 Atari 钥匙碎片，来到一间地下唱片室。这里曾是 A.N. Reverie 存放实体音乐收藏的地方。云端雪崩之后，大多数流媒体曲库消失，只有这些实体媒介仍保留着某种可验证的记忆。

唱机反复播放一段空槽噪声，像是在要求玩家靠近。第二颗星球的核心不是“战胜恐惧”，而是“不要因为恐惧而关掉声音”。

## 4. 核心谜题：遗失的频率

### 4.1 谜题目标

玩家需要在 45 秒内找到三段隐藏频率：

1. 左侧开盘机里的低频；
2. 中央唱针附近的中频；
3. 右侧音箱栅格里的高频。

### 4.2 为什么是“勇气”

这一类谜题给玩家轻微时间压力，要求玩家在不确定和噪声中继续观察。它不惩罚失败，只让玩家重试，重点是模拟“害怕但继续靠近”的动作。

### 4.3 解决后

玩家获得 **Vinyl 钥匙碎片**：

> 这枚碎片象征着勇气。你没有关掉噪声，而是走近它，听见了藏在恐惧背后的旋律。

随后试听室右侧的 **声纹抽屉** 会从静电噪声中显露真正内容，玩家可拾取 **Reverie 的第二段声纹**：

> 不是所有恐惧都需要被战胜，有些只需要被完整听见。

试听室里的 **倒放标签** 和 **唱片排序台** 会共同解锁磁带档案室。玩家把 `DNIWER` 倒读为 `REWIND`，并把唱片按 `1977 → 1978 → 1979 → 1981` 的时间线排列后，可进入档案室拾取 **Reverie 的剪帧索引**：

> 第三把钥匙不在完整影片里，它在被剪掉的 3 分 17 秒。

---

## 5. 扩展谜题：唱片排序

### 5.1 谜题目标

玩家需要按发行年份从早到晚排列四张唱片：

1. `Trans-Europe Express / 1977`
2. `The Man-Machine / 1978`
3. `Unknown Pleasures / 1979`
4. `Computer World / 1981`

### 5.2 失败反馈

排序错误时，Echo-7 会提示：

> 你不是在排喜好榜，你是在复原一段时间线。

该谜题不惩罚失败，只通过角色化反馈引导玩家重新观察年份。

### 5.3 奖励

完成后获得 **Echo-7 的校准片**：

> 顺序不是答案本身，顺序让答案可以被听见。
